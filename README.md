# Network Scanner

A professional Python-based network scanner that discovers active devices on a local network.

## Features
- Network Discovery
- IP & MAC Detection
- Hostname Resolution
- CSV/JSON Export
- Modern CustomTkinter GUI

## Technologies
- Python
- Scapy
- CustomTkinter
- Threading
